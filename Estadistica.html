1.1 Distribucin de Edades

En esta seccin, se analizan las edades de los participantes. Se calculan estadsticas descriptivas como media, mediana y desviacin estndar, y se visualiza un histograma.

# Importar las bibliotecas necesarias
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Cargar los datos
file_path = "Respuestas.xlsx"  # Ruta del archivo
df = pd.read_excel(file_path, sheet_name="Respuestas de formulario 1")

# Configurar el estilo de los grficos
sns.set_style("whitegrid")

# Estadsticas descriptivas de la edad
print("Estadsticas de Edad:")
print(df['Edad'].describe().round(2))

# Histograma de edades
plt.figure(figsize=(10, 6))
plt.hist(df['Edad'], bins=10, edgecolor='black', alpha=0.7, color='skyblue')
plt.title("Distribucin de Edades", fontsize=16, weight='bold')
plt.xlabel("Edad", fontsize=12)
plt.ylabel("Frecuencia", fontsize=12)
plt.grid(axis='y', alpha=0.75)
plt.tight_layout()
plt.show()

2. Niveles de Conocimiento

En esta seccin, se analiza el nivel de conocimiento en diversas tecnologas como Blockchain, Criptomonedas, Inteligencia Artificial (IA), y Machine Learning (ML). Se genera un grfico consolidado que muestra todas las tecnologas en un formato apilado.

In[12]:
# Columnas de tecnologas
knowledge_columns = [
    'Cul es tu nivel de conocimiento sobre Blockchain? ',
    'Cul es tu nivel de conocimiento sobre Criptomonedas? ',
    'Cul es tu nivel de conocimiento sobre Inteligencia Artificial (IA)? ',
    'Cul es tu nivel de conocimiento sobre Machine Learning (ML)? '
]

# Datos de conocimiento
knowledge_data = df[knowledge_columns].apply(pd.Series.value_counts).fillna(0)

# Renombrar las columnas para mayor claridad
knowledge_data.columns = ['Blockchain', 'Criptomonedas', 'IA', 'ML']

# Grfico consolidado
plt.figure(figsize=(12, 8))
knowledge_data.T.plot(kind='bar', stacked=True, colormap='viridis', edgecolor='black')

# Ttulos y etiquetas
plt.title("Distribucin Consolidada de Niveles de Conocimiento por Tecnologa", fontsize=16, weight='bold')
plt.xlabel("Tecnologa", fontsize=12)
plt.ylabel("Cantidad de Respuestas", fontsize=12)
plt.xticks(rotation=0, fontsize=10)

# Leyenda
plt.legend(title="Niveles de Conocimiento", bbox_to_anchor=(1.05, 1), loc='upper left', fontsize=10)

# Ajustar mrgenes
plt.tight_layout()
plt.show()

blockchain_vs_security = pd.crosstab(

blockchain_vs_security.plot(kind='area', figsize=(10, 6), colormap='plasma', alpha=0.8)

plt.xlabel("Niveles de Conocimiento de Blockchain", fontsize=12)
