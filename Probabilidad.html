Problema 2: Probabilidad de estar actualizado y pertenecer a una carrera especfica (Bayes)

# Filtrar datos necesarios
actualizados_bastante = df[df['Qu tan actualizado(a) te consideras en las nuevas tecnologas de sistemas computacionales? '] == 'Bastante']
gestion_software = df[df['Carrera'] == 'Licenciatura en Desarrollo y Gestin del Software']

# Calcular P(A): Proporcin de estudiantes en "Desarrollo y Gestin de Software"
p_a = len(gestion_software) / len(df)

# Calcular P(B): Proporcin de estudiantes bastante actualizados
p_b = len(actualizados_bastante) / len(df)

# Calcular P(B|A): Proporcin de estudiantes bastante actualizados en "Desarrollo y Gestin de Software"
p_b_given_a = len(actualizados_bastante[actualizados_bastante['Carrera'] == 'Licenciatura en Desarrollo y Gestin del Software']) / len(gestion_software)

# Aplicar Teorema de Bayes
p_a_given_b = (p_b_given_a * p_a) / p_b

# Mostrar resultado
print(f"La probabilidad de que un estudiante bastante actualizado pertenezca a 'Desarrollo y Gestin de Software' es: {p_a_given_b:.2%}")

La probabilidad de que un estudiante bastante actualizado pertenezca a 'Desarrollo y Gestin de Software' es: 25.00%

Problema 2: Problema con la distribucion normal

Descripcin:

Este problema utiliza la distribucin normal para modelar las edades de los estudiantes en la encuesta, con una media ( ) de 25 aos y una desviacin estndar ( ) de 5 aos.

In[29]:
import numpy as np
from scipy.stats import norm
import matplotlib.pyplot as plt

# Obtener la columna de edades del DataFrame
edades = df['Edad']

# Parmetros de la distribucin normal basados en los datos reales
mu = edades.mean()  # Media
sigma = edades.std()  # Desviacin estndar

# Lmites ajustados al rango real
min_real = edades.min()
max_real = edades.max()

# Calcular lmites dentro de 2 desviaciones estndar
z_min = max(min_real, mu - 2 * sigma)  # Mximo entre el lmite calculado y el mnimo real
z_max = min(max_real, mu + 2 * sigma)  # Mnimo entre el lmite calculado y el mximo real

# Objetivo 1: Probabilidad dentro del rango "normal" ajustado
probabilidad_normal_ajustada = norm.cdf(z_max, mu, sigma) - norm.cdf(z_min, mu, sigma)

# Objetivo 2: Probabilidad de ser extremo ajustado
probabilidad_extremo_ajustada = 1 - probabilidad_normal_ajustada

# Crear un rango de valores para la grfica
x = np.linspace(mu - 4 * sigma, mu + 4 * sigma, 1000)
pdf = norm.pdf(x, mu, sigma)

# Graficar la distribucin normal ajustada
plt.figure(figsize=(10, 6))
plt.plot(x, pdf, color='blue', label="Distribucin Normal")
plt.axvline(z_min, color='red', linestyle='--', label="Lmite Inferior Ajustado")
plt.axvline(z_max, color='red', linestyle='--', label="Lmite Superior Ajustado")
plt.fill_between(x, pdf, where=(x >= z_min) & (x <= z_max), color='skyblue', alpha=0.5, label="Rango Normal Ajustado")
plt.title("Distribucin Normal de Edades (Ajustada)", fontsize=16, weight='bold')
plt.xlabel("Edad", fontsize=12)
plt.ylabel("Densidad de Probabilidad", fontsize=12)
plt.legend(fontsize=12)
plt.grid(True)
plt.tight_layout()
plt.show()

# Mostrar resultados ajustados
print(f"Probabilidad dentro del rango normal ajustado: {probabilidad_normal_ajustada:.2%}")
print(f"Probabilidad de ser extremo ajustado: {probabilidad_extremo_ajustada:.2%}")

Probabilidad dentro del rango normal ajustado: 80.32%
Probabilidad de ser extremo ajustado: 19.68%
