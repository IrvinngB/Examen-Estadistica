Sobre el Curso

La Estadística se refiere a la colección, análisis, interpretación y presentación de datos. Es aplicable a diversas disciplinas y es crucial para la toma de decisiones. Este curso fortalece habilidades técnicas y blandas, utilizando herramientas tecnológicas para el manejo e interpretación de datos.

Objetivo del Curso

Utilizar técnicas y métodos para clasificar, organizar, representar e interpretar datos de una muestra representativa, utilizando software para la solución de problemas profesionales.

Examen Final
